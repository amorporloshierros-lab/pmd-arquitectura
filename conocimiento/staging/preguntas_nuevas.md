# Preguntas detectadas — Pendientes de revisión

Este archivo se llena AUTOMÁTICAMENTE cada vez que Lucas detecta una
pregunta en una conversación que no coincide con ninguna entrada de
`aprobado/preguntas_frecuentes.md`.

**Qué hacer con esto (1 vez por semana):**

1. Leé cada entrada.
2. Si la pregunta es genuina y puede repetirse → movela a
   `aprobado/preguntas_frecuentes.md` con una respuesta modelo.
3. Si ya existe algo parecido en el aprobado → consolidá y borrá de acá.
4. Si es una pregunta muy específica que no se va a repetir → borrala.
