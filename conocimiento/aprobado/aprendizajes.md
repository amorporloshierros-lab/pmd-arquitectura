# Aprendizajes aprobados — Se inyectan al prompt de Lucas

Acá van tus observaciones estratégicas después de revisar las
conversaciones. No son preguntas-respuestas (eso va en `preguntas_frecuentes.md`),
sino **tácticas de venta, patrones del mercado, cosas que funcionan**.

**Formato recomendado:**

```markdown
## YYYY-MM-DD — Título corto

[Tu observación o aprendizaje]

**Acción para Lucas:** [qué debería hacer diferente a partir de ahora]
```

## 2026-01-15 — El precio funciona mejor si viene después del diferencial

Cuando Lucas arranca la Fase 3 mencionando Entorno Mi Hogar ANTES del número
total, la conversión a Calendly aumenta notoriamente. El orden emocional
"valor → precio" funciona mejor que "precio → valor".

**Acción para Lucas:** el Entorno Mi Hogar va SIEMPRE en el mismo mensaje
que el precio, como justificación inmediata.

---

## 2026-02-03 — Escobar es más sensible al precio que Nordelta

El 70% de leads de Escobar terminan eligiendo Eco Confort o Estándar. Los
de Nordelta van directo a Premium/Élite Lujo.

**Acción para Lucas:** cuando detecto `zona = Escobar`, sugerir primero
Eco/Estándar y enfatizar transparencia (Mi Hogar) como el diferencial —
no competir por precio bajo.
